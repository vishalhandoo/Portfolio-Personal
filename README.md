# Portfolio Personal

A single-page personal portfolio website for Vishal Handoo. The site highlights finance-focused experience, credit risk work, featured projects, certifications, and contact details in a polished static frontend.

## Overview

This portfolio is built as a lightweight static website with expressive visuals and interactive motion. It is designed to present professional background and finance project work in a clean, easy-to-browse format.

## Sections Included

- Hero introduction
- About
- Experience
- Skills
- Projects
- Certifications
- Contact

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Project Structure

```text
.
|-- index.html
|-- style.css
|-- script.js
`-- README.md
```

## Running Locally

Because this is a static site, you can preview it in either of these ways:

1. Open `index.html` directly in your browser.
2. Start a simple local server from the project folder, for example:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customization

- Update page content and section text in `index.html`
- Adjust colors, layout, typography, and responsiveness in `style.css`
- Modify animations and interactivity in `script.js`

## Deployment

This project can be deployed to any static hosting platform, including GitHub Pages.
